## Giới thiệu
Website Shop Bán Giày (Nike Store) được xây dựng hoàn toàn bằng **HTML, CSS và JavaScript thuần**.  
Dự án mô phỏng giao diện và chức năng mua sắm của Nike, giúp người dùng xem sản phẩm, tìm kiếm, xem chi tiết, đăng ký, quản lý profile, giỏ hàng và yêu thích.

## Công nghệ sử dụng
- HTML5
- CSS3 (Responsive + Animation)
- JavaScript thuần
- Material Icons

## Các chức năng chính
- Trang chủ với Hero Banner
- Danh mục sản phẩm và bộ lọc
- Chi tiết sản phẩm (hỗ trợ video)
- Tìm kiếm thông minh
- Đăng ký & Profile thành viên Nike
- Giỏ hàng, Yêu thích, Lịch sử đơn hàng

2. Click đúp vào file `index.html` để chạy bằng trình duyệt
Hoặc dùng **Live Server** trong Visual Studio Code.
