## Giới thiệu dự án

Website Shop Bán Giày (Nike Store) là dự án được xây dựng hoàn toàn bằng **HTML, CSS và JavaScript thuần**.  
Dự án mô phỏng giao diện và một số chức năng chính của website thương mại điện tử Nike, giúp người dùng có thể xem sản phẩm, tìm kiếm, xem chi tiết sản phẩm, đăng ký thành viên, quản lý profile, giỏ hàng và yêu thích.

## Công nghệ sử dụng

- HTML5
- CSS3 (Responsive Design + Animation)
- JavaScript thuần (tương tác người dùng, video player, form validation, localStorage...)
- Material Icons
- Google Fonts

## Cấu trúc thư mục dự án
shopbangiay/
├── index.html
├── assets/
│   ├── css/          ← Các file style.css, product-detail.css...
│   ├── js/           ← Các file JavaScript
│   └── images/       ← Toàn bộ hình ảnh sản phẩm
├── Hưng/             ← Các trang của Hưng (profile, register...)
├── Huy/              ← Cart, Checkout, Favourites...
├── Phúc/             ← Categories, Products...
├── Việt/             ← About, Contact, News...
└── README.md

## Các chức năng chính đã hoàn thành

- Trang chủ (Hero banner + Slider)
- Danh mục sản phẩm & Bộ lọc (categories.html)
- Chi tiết sản phẩm (ảnh + video, chọn size, Add to Bag)
- Tìm kiếm thông minh
- Đăng ký & Đăng nhập thành viên
- Trang Profile (Interests, Member Benefits, Nike Apps)
- Giỏ hàng, Yêu thích, Lịch sử đơn hàng
- Trang Tin tức, Khuyến mãi, Về chúng tôi, Liên hệ

## Hướng dẫn chạy dự án

1. Tải source code về máy
2. Mở thư mục dự án
3. Click đúp vào file `index.html` để mở bằng trình duyệt (Chrome khuyến nghị)

Hoặc mở bằng Live Server trong Visual Studio Code.
